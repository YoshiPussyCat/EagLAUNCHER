# EAGLAUNCHER

A launcher created to improve your gameplay with EaglerCraft, an open-source minecraft on browsers.

------

## Project Overview

EagLAUNCHER is a web-based launcher that allows users to access Eaglercraft versions through a clean and interactive interface, without requiring downloads or installations.
It features many EaglerCraft official versions and modded clients. A huge inspiration from irv77 for this project.
